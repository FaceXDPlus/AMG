# AMG 
- A real open-source software for processing face tracking data and rendering on PC.
- 一款真开源的面捕PC渲染端
- LTS/长期支持

## Feature/特性
* Multi language support/多语言支持
* Multi Model & Source support/多模型&输入源支持
* Multi-player support(Powered by WS)/多用户支持(使用WS实现)
